# 📌 Obligatorio 1 – Desarrollo Full Stack

Aplicación web desarrollada como parte del curso de Desarrollo Full Stack, que permite la gestión de entidades mediante una arquitectura cliente-servidor, integrando frontend, backend y base de datos.

---

## 🚀 Descripción

Este proyecto consiste en el desarrollo de una aplicación web completa que permite la gestión de información a través de una API REST. Se implementa una arquitectura en capas, separando responsabilidades entre frontend, backend y persistencia de datos.

La aplicación incluye funcionalidades CRUD, validaciones y comunicación entre cliente y servidor mediante HTTP.

---

## 🧱 Arquitectura

El sistema está dividido en:

- **Frontend**: Interfaz de usuario interactiva y responsiva  
- **Backend**: API REST encargada de la lógica de negocio  
- **Base de Datos**: Persistencia de la información  

Se aplica separación de responsabilidades y buenas prácticas para mantener el código modular y escalable.

---

## ⚙️ Funcionalidades

- Operaciones CRUD completas (Crear, Leer, Actualizar, Eliminar)  
- Consumo de API REST desde el frontend  
- Validación de datos en cliente y servidor  
- Manejo de errores y respuestas HTTP  
- Navegación e interacción dinámica en la interfaz
