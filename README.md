# 📌 Obligatorio 1 – Desarrollo Full Stack (API REST)

API REST desarrollada como parte del curso de Desarrollo Full Stack, encargada de la lógica de negocio, gestión de datos y exposición de endpoints para ser consumidos por aplicaciones cliente (frontend).

---

## 🚀 Descripción

Este proyecto consiste en el desarrollo de una API REST que permite la gestión de entidades mediante endpoints accesibles a través de HTTP.

La API implementa operaciones CRUD, validaciones y persistencia de datos, siguiendo una arquitectura en capas y buenas prácticas de desarrollo backend.

Una API REST permite la comunicación entre cliente y servidor mediante solicitudes HTTP (GET, POST, PUT, DELETE), facilitando el intercambio de datos entre sistemas :contentReference[oaicite:0]{index=0}.

---

## 🧱 Arquitectura

La aplicación está estructurada en capas para garantizar escalabilidad y mantenibilidad:

- **Controllers**: Manejo de endpoints y solicitudes HTTP  
- **Services / Lógica de negocio**: Procesamiento de reglas del sistema  
- **Persistencia**: Acceso a base de datos
- **Modelos**: Representación de entidades del dominio  

Se aplica separación de responsabilidades siguiendo el modelo cliente-servidor.
